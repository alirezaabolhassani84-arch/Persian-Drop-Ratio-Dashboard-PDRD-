# Persian-Drop-Ratio-Dashboard-PDRD-
سیستم داشبورد خودکار که هر ۸ ساعت داده‌های Triggered، CSM و Pardis را تحلیل کرده و گزارش کامل با ۷ بخش اصلی (KPI، Drop Ratio، آمار، نمودارها و هشدارها) تولید می‌کند. با طراحی حرفه‌ای و رنگ‌بندی اختصاصی (آبی/قرمز/سبز) از طریق ایمیل ارسال شده و سرویس‌های با Drop بالا را هوشمندانه هشدار می‌دهد.
